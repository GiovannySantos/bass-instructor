# Bass Dojo - Mestre do Groove v3

App de prática para baixistas com fretboard inteligente, flashcards, metronômio e briefing de gig.

## Stack
- React + Vite
- Tailwind CSS
- Tone.js
- PWA (manifest + service worker)

## Scripts
```bash
npm install
npm run dev
npm run build
npm run preview
```

## PWA
Arquivos estáticos ficam em `public/`. O service worker básico está em `public/sw.js`.
